# 小芽家教 (SproutChat) PRD

## 1. 产品概述

### 1.1 产品名称
小芽家教 (SproutChat)

### 1.2 产品定位
面向一年级学生的 AI-First 个性化家教助手

### 1.3 产品愿景
让每个孩子都能拥有一个懂他们、会引导的 AI 小老师，在快乐中学习成长

## 2. 核心功能

### 2.1 目标用户
- **主要用户**：一年级学生（6-7岁）
- **次要用户**：家长、小学教师

### 2.2 核心价值主张
- 语音拍照交互，降低使用门槛
- 引导式教学，培养思考能力
- 个性化学习路径，因材施教

### 2.3 核心功能矩阵

| 功能模块 | 功能描述 | 优先级 |
|---------|---------|--------|
| 语音交互 | 自然语音对话，识别儿童语言 | P0 |
| 拍照识别 | 拍摄题目，自动识别内容 | P0 |
| 引导式教学 | 苏格拉底式提问，不给答案 | P0 |
| 学习进度 | 记录学习轨迹，生成报告 | P1 |
| 家长模式 | 学习报告查看，设置管理 | P1 |

## 3. 用户体验设计

### 3.1 交互方式
- **主要交互**：语音对话
- **辅助交互**：拍照上传
- **应急交互**：简单文字输入

### 3.2 视觉设计
- **主题色**：嫩绿色 (#8BC34A)
- **辅助色**：天空蓝 (#64B5F6)
- **字体**：圆润可爱的儿童字体

### 3.3 使用场景
1. 作业辅导：不会做的题目拍照上传
2. 知识学习：主动询问不懂的知识点
3. 复习巩固：通过对话回顾所学内容

## 4. 技术架构

### 4.1 核心技术栈
- **后端**：Python FastAPI
- **前端**：React + Tailwind CSS
- **AI模型**：Claude API (教育场景优化)
- **语音**：Web Speech API + 云端ASR

### 4.2 关键技术挑战
- 儿童语音识别准确率
- 题目图像理解和解析
- 教育对话生成质量

## 5. 产品指标

### 5.1 核心指标
- **活跃度**：日活跃用户数 (DAU)
- **粘性**：用户留存率 (次日、7日、30日)
- **效果**：学习提升度 (通过前后测对比)
- **满意度**：NPS 净推荐值

### 5.2 关键节点
- **MVP**：基础语音交互 + 简单数学题引导
- **V1.0**：完整功能 + 家长模式
- **V2.0**：多科目支持 + 个性化推荐

## 6. 风险与限制

### 6.1 技术风险
- 语音识别在嘈杂环境下的准确率
- AI 模型生成内容的准确性把控
- 响应延迟影响用户体验

### 6.2 产品风险
- 儿童屏幕时间管理
- 家长对 AI 教育的接受度
- 与传统教育的结合度

## 7. 版本规划

### 7.1 MVP (0-2个月)
- 基础语音交互
- 数学题目拍照识别
- 简单引导式对话

### 7.2 V1.0 (2-4个月)
- 完整家长模式
- 学习进度追踪
- 多题型支持

### 7.3 V2.0 (4-6个月)
- 语文/英语科目扩展
- 个性化学习路径
- 同伴学习功能
